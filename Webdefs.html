Internet and Web Definitions

By Takuto Yamashiro

Internet

KA

The internet is a worldwide network of connected computers, phones, and smart devices that communicate with each other to quickly exchange information and provide services.

Web

a network of fine threads constructed by a spider from fluid secreted by its spinnerets, used to catch its prey.

How are the internet and Web different

The internet is a global network of computers that links people, whereas the web is the system that enables users to retrieve information from the internet.

ASCII and Unicode

ASCII is a code for English letters and symbols, while Unicode is a code for letters, symbols, and emojis from all languages.

What is UTF-8

UTF-8 is a way of turning characters into numbers so computers can use many languages and emojis.

URL

A URL (Uniform Resource Locator) is the address used to find a website or page on the internet.

Domain Name

A domain name is the website address you enter in a browser, such as "google.com."

HTTP

Hypertext Transfer Protocol (HTTP) is a set of rules that govern how web browsers and servers exchange information. It's the foundation of the World Wide Web, and is used to load web pages.

HTTPS

Hypertext Transfer Protocol Secure (HTTPS) is used for transfering data over the web, and it also add a layer of security by encrypting the data.

IP Address

An IP address is a unique number given to each device on a network to enable communication with other devides.

DNS

DNS (Domain Name System) is a system that converts domain names, such as "google.com," into IP addresses to help computers find websites.

W3C Recommendations

W3C Recommendations are official standards set by the World Wide Wev Consortium (W3C) to ensure web technologies work consistently and are accessible across the internet.

XML, XHTML, HTML, HTML5

XML is a flexible format for storing data with custom tags, XHTML is a stricter version of HTML that follows XML rules, HTML is the standard language for creating web pages, and HTML5 os the latest version offering new features for multimadia, accessibility, and performance.
